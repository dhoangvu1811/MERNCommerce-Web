# OrderTable Components

Thư mục này chứa các components con được phân chia từ OrderTable.jsx để tăng tính modular và tái sử dụng.

## Cấu trúc Components:

- **OrderDataGrid.jsx**: DataGrid chính hiển thị danh sách đơn hàng
- **OrderSearchBar.jsx**: Thanh tìm kiếm đơn hàng
- **OrderTableHeader.jsx**: Header với title và search bar
- **OrderDetailsDialog.jsx**: Dialog hiển thị chi tiết đơn hàng
- **index.js**: Export tất cả components để import dễ dàng

## Mục đích:

- Tăng tính tái sử dụng
- Dễ bảo trì và test
- Code rõ ràng, dễ đọc
- Tuân thủ nguyên tắc Single Responsibility
